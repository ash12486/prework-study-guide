HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 h2 h3 h4 h5 h6 tags represents header font sizes.
  • The li tags represents bullet points.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A background-color displays the color of the background. This can be entered as 'Blue' or Hex#

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull command origin main downloads new changes from the branch named main on the remote named origin and integrates them into your local HEAD branch
  • git clone retrieves an entire repository from a hosted location via URL
